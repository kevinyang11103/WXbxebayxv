# 基于SpringBoot的JAVA学习系统 python1168

## 技术概述

本项目采用以下技术栈进行开发：

- **后端语言**：Java
- **开发框架**：SpringBoot
- **数据库**：MySQL
- **架构模式**：B/S

## 功能模块

### 前台功能模块

- **首页功能**：
  - 查看系统首页、课程信息、作业信息、资料信息等。
- **用户注册与登录**：
  - 注册账号、登录系统、查看和管理个人信息。
- **课程信息查看与操作**：
  - 查看课程详情、进行评论和收藏操作。
- **作业信息查看与操作**：
  - 查看作业详情、收藏作业。
- **资料信息查看与操作**：
  - 查看资料详情、点赞和评论。
- **个人中心**：
  - 更新个人信息、管理发布内容、收藏、考试记录、错题本等。

### 后台功能模块

- **管理员功能**：
  - 管理学生、教师、课程、作业、资料、签到、成绩、试卷、试题、校园论坛、系统信息等。
- **学生功能**：
  - 管理签到、作业、论坛、收藏、考试等。
- **教师功能**：
  - 管理课程、作业、资料、签到、成绩、学生作业等。

## 系统角色

系统包含以下角色：

- **学生**
- **教师**
- **管理员**

## 运行环境

- **操作系统**：支持主流操作系统
- **数据库**：MySQL（建议版本 5.7+）
- **Java环境**：JDK 1.8+

1. 准备所需环境（Java、MySQL等）。

3. 部署应用服务。
